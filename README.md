# Neirix — Telegram-бот коллега-помощник

Бот для рабочих чатов, который автоматически распознаёт задачи и встречи из обычных сообщений, помогает управлять ими, отправляет напоминания и генерирует саммари обсуждений.

---

## 🧠 Как работает

### Распознавание из сообщений
Бот анализирует каждое сообщение в чате с помощью:
- **[all-MiniLM-L6-v2](https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2)** — модель встраивания (embeddings) для семантического анализа
- **GigaChat API** — для извлечения структурированных данных (задачи, встречи, даты, ответственные)

Бот автоматически определяет:
- **Задачи** — поручения, дедлайны, ответственных
- **Встречи** — созвоны, совещания, время, участников

### Примеры распознавания

| Сообщение | Что бот извлечёт |
|---|---|
| `завтра у нас встреча с заказчиком в 9` | 📅 Встреча: "встреча с заказчиком", завтра 09:00 |
| `задача для @JDexMoment - сделать бота к субботе` | 📋 Задача: "сделать бота", @JDexMoment, до субботы |
| `к воскресенью @user1 и @user2 должны сделать тест` | 📋 Задача: "сделать тест", @user1 + @user2, до воскресенья |
| `у @user1 и @user2 5 мая в 11:30 будет встреча с директором` | 📅 Встреча: "встреча с директором", 05.05 11:30, @user1 + @user2 |

## 📋 Управление задачами

### Команды
| Команда | Описание |
|---|---|
| `/tasks` | Список открытых задач |

### Возможности
- Автоматическое извлечение задач из сообщений
- Поддержка нескольких ответственных (`@user1 и @user2`)
- Распознавание дедлайнов в любом формате:
  - `завтра`, `послезавтра`
  - `к субботе`, `в это воскресенье`, `к следующему вторнику`
  - `25 числа`, `5 мая`, `25.05.2026`
- Кнопка **✅ Выполнено** под каждой задачей
- Проверка прав: закрыть задачу может только назначенный ответственный

---

## 📅 Управление встречами

### Команды
| Команда | Описание |
|---|---|
| `/meetings` | Список предстоящих встреч |

### Возможности
- Автоматическое извлечение встреч из сообщений
- Поддержка нескольких участников
- Распознавание даты и времени в любом формате
- Кнопки управления под каждой встречей:
  - **📅 Перенести** — бот запрашивает новую дату через FSM
  - **❌ Отменить** — с подтверждением
- При переносе:
  - Участники получают уведомление о новом времени
  - Напоминания сбрасываются и отправятся заново
- При отмене:
  - Участники получают уведомление
  - Встреча исчезает из списка

## 📊 Саммари обсуждений

### Команды
| Команда | Описание |
|---|---|
| `/summary today` | Саммари за сегодня |
| `/summary yesterday` | Саммари за вчера |
| `/summary week` | Саммари за прошлую неделю |
| `/summary YYYY-MM-DD YYYY-MM-DD` | Саммари за произвольный период |

### Что включает саммари
- 📋 Основные темы обсуждения
- ✅ Принятые решения
- 📌 Задачи и ответственные (с кликабельными ссылками на профили)
- 📅 Запланированные встречи (с участниками)
- ❓ Нерешённые вопросы

### Особенности
- Кэширование: повторный запрос за тот же период возвращает готовое саммари
- Контекст из векторного хранилища (Qdrant) — релевантные предыдущие обсуждения
- Разбиение длинных саммари на несколько сообщений
- Ссылки на пользователей: `@username` или HTML-ссылка на профиль

## 🔔 Система напоминаний

Бот отправляет напоминания в **личные сообщения** участников.

| Тип напоминания | Когда отправляется | Дубли |
|---|---|---|
| Встреча через 24 часа | Каждые 30 мин проверка | ❌ Нет (флаг `daily_reminder_sent`) |
| Встреча через 1 час | Каждые 5 мин проверка | ❌ Нет (флаг `reminder_sent`) |
| Задача: дедлайн через 24 часа | Каждые 30 мин проверка | ❌ Нет (флаг `daily_reminder_sent`) |
| Задача просрочена | 1 раз в день (10:00) | ❌ Нет (флаг `overdue_reminder_sent`) |
| Утренний дайджест | Ежедневно в 9:00 | ✅ Каждый день |

### Утренний дайджест включает
- Задачи на сегодня
- Встречи на сегодня

### Фильтрация
- Боты (`*_bot`, `*Bot`) автоматически исключаются из уведомлений

## 🔗 Привязка чата

### Команды
| Команда | Описание |
|---|---|
| `/link_chat` | Получить код привязки (только в групповых чатах) |

### Как работает
1. Добавьте бота в групповой чат
2. Бот автоматически отправит код привязки
3. Или выполните `/link_chat` для получения кода
4. Отправьте код боту в личные сообщения
5. Ваш аккаунт привяжется к чату — вы будете получать уведомления

## ⚙️ Технический стек

| Компонент | Технология |
|---|---|
| Telegram Bot | aiogram 3.x |
| Backend / ORM | Django |
| LLM | GigaChat API (бесплатный) |
| Embeddings | sentence-transformers/all-MiniLM-L6-v2 |
| Векторное хранилище | Qdrant |
| Периодические задачи | Celery + Celery Beat |
| Брокер сообщений | Redis |
| База данных | PostgreSQL |
| Контейнеризация | Docker + Docker Compose |

## 🧪 Тестирование

```bash
# Запуск всех тестов
pytest

# Запуск конкретного файла
pytest tests/bot/test_tasks.py -v

# Запуск с подробным выводом
pytest -v -s
```

### Что покрыто тестами
- Команды `/tasks`, `/meetings`, `/summary`
- Callback-кнопки (выполнить задачу, отменить/перенести встречу)
- Привязка чата (`/link_chat`, обработка кода)
- Добавление бота в чат
- Напоминания (Celery-задачи)
- Отправка уведомлений (NotificationSender)
- Фильтрация ботов
- Граничные случаи: пустые данные, длинные строки, HTML-инъекции, невалидные даты

## 📌 Особенности

- **Ленивая загрузка ML-моделей** — SentenceTransformer и GigaChat загружаются только при первом использовании, не при импорте
- **Детерминированный парсинг дат** — относительные даты (`к субботе`, `в это воскресенье`) вычисляются на Python, а не доверяются LLM
- **Фильтрация ботов** — боты автоматически исключаются из ответственных, участников и уведомлений
- **FSM для переноса встреч** — надёжное управление состоянием с таймаутом
- **Флаги напоминаний** — каждое напоминание отправляется ровно один раз
- **Сброс напоминаний при переносе** — при изменении даты встречи напоминания отправятся заново
